# History Clock

Turns a blank github history into a clock. Can be refreshed.

You must first specify your credentials in the following 3 files:

- config/myemail

- config/mygithubname

- config/myoauthtoken

The format is specified in the provided example files.

### Usage:
```bash
./run.sh
```
